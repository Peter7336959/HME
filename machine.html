危險性機械

Important定義？
  • 危險性機械是指符合中央主管機關所定一定容量以上之機械。這類機械因構造複雜、作業風險高，非經檢查合格不得使用。

  • 根據：《職業安全衛生法施行細則》第 22條及《危險性機械及設備安全檢查規則》第 3 條之規定。

    ⁕ 危險性機械的種類、容量定義及其具體型式如下：

起重機具安全試驗

荷重試驗 額定荷重 × 1.25 (200t 以上為額定+50t) 驗證構造強度，需作吊升、旋轉、走行等動作。
安定度試驗 額定荷重 × 1.27 驗證翻覆極限，需在最不利安定條件下進行。

  • 定義：這是在特定場所使用動力將貨物吊升並作水平搬運的機械。

  • 容量： 吊升荷重在 3 公噸以上之固定式起重機，或 1 公噸以上之斯達卡式起重機（駕駛室隨貨物升降者）。

    • 吊升荷重？ 額定荷重？相關安全試驗？
吊升/ 額定荷重？(標示)

  • (1) 固定式起重機（未具伸臂者）：

    • 額定荷重 = 吊升荷重 － 吊具重量（如吊鉤、抓斗）

  • (2) 具有伸臂之起重機（如移動式或伸臂固定式）：

    • 額定荷重 = (特定作業半徑/角度下足以承受之最大荷重) － 吊具重量 (註：此時的「吊升荷重」僅指在最有利條件下之最大額定荷重加吊具重)
  • 主要型式：

    • 架空式起重機： 包括普通型及特殊型（如迴轉載人吊運車、斯達卡式起重機）。

    • 伸臂型起重機： 包括塔型、門型、低台、立柱、升高及鎚頭型等。

    • 橋型起重機： 包括普通橋型及特殊橋型。

    • 卸載機 (Unloaders)： 包括普通型、載人吊運車型及平動卸載機。

    • 其他： 纜索起重機、貨櫃起重機及單軌起重機。

  • 特別註記： 鑒於近年工安事故，設置於營建工地之塔式起重機（塔吊），不論吊升荷重大小，皆已全面納入危險性機械管理。（115/12/23生效）

  • 指能自行移動於非特定場所，且具有起重動力之起重機。

  • 容量定義： 吊升荷重在 3 公噸以上之移動式起重機。

  • 主要型式：

    • 卡車起重機： 分為伸臂伸縮式與不伸縮式。

    • 輪行起重機： 包含全路面、傳統式等。

    • 履帶起重機： 接地面積廣，適用於軟弱地盤。

    • 鐵路起重機： 包含救援起重機及桁樑架起重機。

  • 註： 為了防止墜落風險，具有搭乘設備供乘載或吊升人員之移動式起重機（載人吊車），不論噸數皆列為危險性機械管理。（115/12/23生效）

  • 以動力吊升貨物為目的，具有主柱及吊桿，並以鋼索操作升降之機械。

  • 容量定義： 吊升荷重在 3 公噸以上者。

  • 主要型式： 單柱、雙柱、牽索及定肢人字臂起重桿。

  • 圖片：人字臂起重桿

  • 設置於營建工地，供營造施工使用，搬器順沿軌道鉛直升降之機械。

  • 容量定義： 不限積載荷重，凡供營造施工使用者皆屬之。

  • 主要型式： 包括鋼索式、液壓式及齒條式升降機，以及長跨度工程用升降機。

  • 圖片：營建用升降機 Ｑ：Mast Climbing Work Platform (MCWP)？

  • 於土木、建築工程中，僅以搬運貨物為目的之升降機。

  • 容量定義： 導軌或升降路高度在 20 公尺以上者。

  • 主要型式： 包括塔式、雙柱式、單柱式及雙側式提升機。

  • 由懸吊式施工架、升降裝置、工作台等構成，供人員升降施工之設備。

  • 容量定義： 不限容量，凡載人用吊籠皆列管。

  • 主要型式：

    • 常設型： 吊臂固定型、吊臂俯仰型、單軌懸垂型。

    • 可搬型： 懸臂型、支架懸垂型及椅式吊籠。

  • 圖示：吊籠

  • 目前法規列明上述六大類為核心管理對象，若未來有新型態高風險機械，中央主管機關得另行指定公告為危險性機械。

  • 總結來說，危險性機械的管理重點在於「一定容量以上」或「載人使用」的起重升降設備。為了確保安全，這些機械必須經過型式檢查、竣工檢查（或使用檢查）及定期檢查合格後，方能繼續使用。

危險性機械 (Hazardous Machines)

種類 (大分類) 型式 (包含中/小分類) 容量分級基準
固定式起重機 架空式、伸臂起重機、橋型起重機、卸載機、纜索起重機、貨櫃起重機、單軌起重機 吊升荷重 (t)：1. 未滿 50t / 2. 50t ~ 未滿 100t / 3. 100t 以上
移動式起重機 卡車起重機、輪行起重機 (拖車起重機)、履帶起重機、鐵路起重機 吊升荷重 (t)：1. 10t 以下 / 2. 超過 10t ~ 未滿 100t / 3. 100t 以上
人字臂起重桿 單柱、雙柱、牽索、定肢人字臂起重桿 同固定式起重機之分級
升降機 鋼索式、液壓式、齒條式、工程用升降機、長跨度工程用升降機 積載荷重 (t)：1. 未滿 3t / 2. 3t 以上
營建用提升機 塔式、雙柱式、單柱式、雙側式、長跨度提升機 導軌或升降路高度 (m)
吊籠 常設型 (吊臂固定/俯仰、單軌懸垂)<br>可搬型 (懸臂、支架懸垂、椅式) 依走行形式、工作台形式及吊臂運動方式組合

說明與備註

1. 起重機互認機制： 具有「吊運車式」製造許可者可製造「吊重器式」；具有「架空式」許可者可製造「單軌起重機」，無需另外申請。

2. 型式細分： 吊籠的型式是由其走行形式（如軌道式）、工作台形式（如具有升降裝置的操作台式）與吊臂運動（如旋轉）區分並組合而成。
